INSTAGRAM LOGIN EXAMPLE
=======================


Configuration
------------
Create a *credentials.json* file like this:

```
{
  "INSTAGRAM_CLIENT_ID"     : "--insert-instagram-client-id-here--",
  "INSTAGRAM_CLIENT_SECRET" : "--insert-instagram-client-secret-here--"
}

```

Installation
------------
```
npm install

```

Usage
------------
```
npm start

```
